pipe: dataTableSearchFilter

Usage

array | dataTableSearchFilter : expression : targets

Arguments

Param Type Details
expression The string to search.
targets Array of objects that is used to filter specific properties. For example [{searchable: true, targets: 'email'}, {searchable: true, targets: 'name'}].
